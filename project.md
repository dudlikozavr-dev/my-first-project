# Проект: Шелковые халаты

## Что это за проект

Интернет-витрина для продажи атласных халатов с доставкой. Проект создан как набор статических HTML-страниц без внешних зависимостей — всё работает прямо из браузера. Целевая аудитория привлекается через Telegram (@Dalenk) и лендинг.

## Для кого

- Покупатели: женщины, которые ищут стильный и доступный атласный халат
- Владелец: Денис Дмитриев — онлайн предприниматель, управляет магазином и коммуникацией через Telegram

## Страницы (уже готово)

| Файл | Описание |
|------|----------|
| `vizitka.html` | Персональная визитка Дениса Дмитриева с фото, специализацией и кнопкой «Написать мне» |
| `landing.html` | Лендинг шелковых халатов: hero, 3 преимущества, галерея из 5 фото, блок «Для кого», CTA |

## Страницы (в планах)

| Файл | Описание |
|------|----------|
| `catalog.html` | Каталог — 5 халатов с фото, названием, ценой и кнопкой «Заказать» |
| `order.html` | Страница оформления заказа — форма с выбором халата, размера, контактов |
| `reviews.html` | Отзывы покупателей |
| `about.html` | О компании — история, ценности, фото |
| `delivery.html` | Условия доставки, оплаты и возврата товара |

## Дизайн

- **Стиль:** тёмный премиум (dark luxury)
- **Фон:** `#0d0a0f` с анимированными градиентными пятнами
- **Акцент:** золото (`#d4a843`, `#f5d483`) + розово-пурпурный (`#c9748a`)
- **Типографика:** системный шрифт (`-apple-system`, `Segoe UI`)
- **Компоненты:** стеклянные карточки (glassmorphism), градиентные заголовки, hover-анимации, scroll-reveal
- **Иконки:** inline SVG, без внешних библиотек
- **Адаптивность:** все страницы корректно отображаются на мобильных и десктопе
