## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта

Убедить посетительницу написать в Telegram, чтобы оформить заказ.

Лендинг не продаёт напрямую — он **снимает достаточно страхов**, чтобы человек решился на первый контакт. Продажа завершается в живом разговоре с Денисом.

### Целевая аудитория

**Основная:** женщины 25–42 лет, Россия, покупают онлайн.

| Психотип | Триггер | Что ей важно |
|---|---|---|
| Занятая мама | «Моё время для себя» | Быстро, понятно, можно вернуть |
| Городской профессионал | Качество и эстетика | Выглядит дорого, сделано хорошо |
| Ценительница уюта | Сенсорный комфорт | Максимально мягко, приятно на коже |
| Дарительница | Подарок с wow-эффектом | Красивая упаковка, точно понравится |

**Вторичная:** мужчины 25–45, ищут подарок для женщины.

### Ключевое сообщение

> «Твоё утро начинается здесь» — продаём ритуал и ощущение, а не атласный халат.

## 4. ДИЗАЙН

### Настроение и стиль

**Dark romantic luxury** — тёмный фон, золотые акценты, ощущение японской эстетики и домашней роскоши. Ориентир: Coco de Mer + Deshabille.

Не «дорого и недоступно» — а «ты это заслуживаешь, и это реально купить».

### Цветовая палитра

```css
--bg:      #0d0a0f;  /* основной фон */
--surface: #1a1420;  /* фон карточек, glassmorphism */
--gold:    #d4a843;  /* главный акцент */
--rose:    #c9748a;  /* второй акцент */
--cream:   #f0e8d8;  /* основной текст (не белый!) */
--blush:   #e8b4b8;  /* нежные детали, иконки */
--mauve:   #9c6b7a;  /* приглушённые подписи */
--plum:    #3d1a2e;  /* третий цвет орба */
--muted:   rgba(240,232,216,0.5); /* вторичный текст */
```

## 6. АНИМАЦИИ

### Обязательные (без них лендинг «мёртвый»)

**1. Scroll-reveal для всех секций**
```js
const io = new IntersectionObserver(
  entries => entries.forEach(e => e.isIntersecting && e.target.classList.add('visible')),
  { threshold: 0.15 }
);
document.querySelectorAll('.reveal').forEach(el => io.observe(el));
```
```css
.reveal { opacity: 0; transform: translateY(40px); transition: 0.7s ease; }
.reveal.visible { opacity: 1; transform: translateY(0); }
```

**2. Дышащие орбы**
```css
@keyframes pulse-orb {
  0%, 100% { transform: translate(-50%,-50%) scale(1);    opacity: 0.18; }
  50%       { transform: translate(-50%,-50%) scale(1.15); opacity: 0.25; }
}
.orb { animation: pulse-orb 8s ease-in-out infinite; }
```

**3. Shimmer на hero-заголовке**
```css
@keyframes shimmer {
  0%   { background-position: -200% center; }
  100% { background-position:  200% center; }
}
```

### Рекомендуемые (усиливают впечатление)

**4. Floating image (если есть фото халата в hero)**
```css
@keyframes float {
  0%, 100% { transform: translateY(0) rotate(-1deg); }
  50%       { transform: translateY(-12px) rotate(1deg); }
}
.hero-img { animation: float 6s ease-in-out infinite; }
```

**5. Card shine при hover (spotlight под курсором)**
```js
card.addEventListener('mousemove', e => {
  const r = card.getBoundingClientRect();
  card.style.setProperty('--mx', ((e.clientX-r.left)/r.width*100)+'%');
  card.style.setProperty('--my', ((e.clientY-r.top)/r.height*100)+'%');
});
```
```css
.card::after {
  content: '';
  position: absolute; inset: 0;
  background: radial-gradient(circle at var(--mx,50%) var(--my,50%),
    rgba(255,255,255,0.05), transparent 60%);
  pointer-events: none;
}
```

**6. Stagger для карточек галереи**
```css
.card:nth-child(1) { transition-delay: 0s; }
.card:nth-child(2) { transition-delay: 0.1s; }
.card:nth-child(3) { transition-delay: 0.2s; }
```

**7. Animated gradient border на финальной CTA**
```css
@property --angle { syntax: '<angle>'; initial-value: 0deg; inherits: false; }
@keyframes spin-border { to { --angle: 360deg; } }
.btn-telegram {
  background:
    linear-gradient(#0d0a0f, #0d0a0f) padding-box,
    conic-gradient(from var(--angle), #d4a843, #c9748a, #f59e0b, #d4a843) border-box;
  border: 2px solid transparent;
  animation: spin-border 4s linear infinite;
}
```

**8. Marquee между секциями (декоративно)**
```css
@keyframes marquee { from { transform: translateX(0); } to { transform: translateX(-50%); } }
.marquee { animation: marquee 20s linear infinite; white-space: nowrap; overflow: hidden; }
```
Текст: «· Халаты кимоно · Доставка по России · Оплата при получении · Твой ритуал ·»

## 7. АДАПТИВНОСТЬ

### Breakpoints

```css
/* Мобильный (основа) */
/* Планшет */
@media (min-width: 768px) { ... }
/* Десктоп */
@media (min-width: 1024px) { ... }
/* Широкий */
@media (min-width: 1440px) { ... }
```

### Правила

| Элемент | Мобильный | Планшет | Десктоп |
|---|---|---|---|
| Карточки галереи | 1 колонка | 2 колонки | 3 колонки |
| Строка доверия | 2×2 | 4 в ряд | 4 в ряд |
| Hero H1 | ~2.2rem | ~3.5rem | ~4.5rem |
| Тезисы материала | 1 колонка | 2 колонки | 4 в ряд |
| Отзывы | scroll-snap | 2 колонки | 3 колонки |
| Sticky CTA | есть | скрыт | скрыт |
