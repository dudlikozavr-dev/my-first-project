## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта

Дать покупательнице возможность **выбрать халат, добавить в корзину и оплатить онлайн** без звонков и ожидания. Консультант Елена доступна в Telegram и Max для вопросов до и после покупки.

Сайт — полноценный мини-магазин на 5 позиций: каталог → корзина → оформление заказа → оплата через ЮKassa.

### Целевая аудитория

**Основная:** женщины 25–42 лет, Россия, покупают онлайн.

| Психотип | Триггер | Что ей важно |
|---|---|---|
| Занятая мама | «Моё время для себя» | Быстро оформить, оплатить картой, получить |
| Городской профессионал | Качество и эстетика | Выглядит дорого, сделано хорошо |
| Ценительница уюта | Сенсорный комфорт | Максимально мягко, приятно на коже |
| Дарительница | Подарок с wow-эффектом | Красивая упаковка, доставка в срок |

**Вторичная:** мужчины 25–45, ищут подарок для женщины.

### Ключевое сообщение

> «Твоё утро начинается здесь» — продаём ритуал и ощущение, а не атласный халат.

### 10 — КОРЗИНА (плавающий элемент)

**Тег:** `<div id="cart-panel">` — не секция, а фиксированный UI-элемент

**Иконка корзины** — фиксированная, правый нижний угол или правый верхний.
- Показывает количество товаров: badge с числом
- При клике — открывается боковая панель (drawer) справа

**Боковая панель корзины:**
- Список добавленных товаров: фото + название + размер + цена
- Кнопка удалить (×) для каждой позиции
- Итого: сумма
- Кнопка «Оформить заказ» → переход к секции [11]
- Кнопка «Продолжить покупки» → закрывает панель

**Состояние хранится в:** `localStorage` (без регистрации, без сервера)

### Виджет ЮKassa

```html

  confirmation_token: 'ТОКЕН_С_СЕРВЕРА', // получить через API ЮKassa
```

> **Важно:** ЮKassa требует серверную часть для создания платежа и получения confirmation_token. Это нельзя реализовать в одном HTML-файле без бэкенда. Варианты:
> - **Вариант A:** Простой PHP-файл `pay.php` на том же хостинге
> - **Вариант B:** Netlify Functions / Vercel Function (serverless)
> - **Вариант C:** Готовое решение на Tilda / Ecwid с интеграцией ЮKassa

### Обязательные

**Scroll-reveal:**
```js
const io = new IntersectionObserver(
  entries => entries.forEach(e => e.isIntersecting && e.target.classList.add('visible')),
  { threshold: 0.15 }
);
document.querySelectorAll('.reveal').forEach(el => io.observe(el));
```
```css
.reveal { opacity: 0; transform: translateY(40px); transition: 0.7s ease; }
.reveal.visible { opacity: 1; transform: translateY(0); }
```

```css
@keyframes pulse-orb {
```

### Ограничения HTML/CSS/JS

- CSS внутри `<style>`, JS внутри `<script>`
- Внешние библиотеки: только Google Fonts + ЮKassa widget
- Иконки: только inline SVG
- Корзина: `localStorage` (без базы данных)
- Минимальная ширина: 320px

### Ограничение ЮKassa

Создание платежа через ЮKassa **требует серверный запрос** с секретным ключом. Чистый HTML без сервера не подойдёт. Нужно выбрать один из вариантов (см. раздел 6).
